Embed the Impacts Foundry widget v1

Drop a script + a <div> into any page on your site to show your pinned Impacts Foundry projects with a one-click supporter pathway. The widget renders inside a Shadow DOM so your site's CSS will not be affected (or affected by it).

1. Minimum embed

<div data-gmi-widget data-api-key="YOUR_API_KEY"></div>
<script src="https://www.impacts-foundry.com/app/widget/v1/embed.js" async></script>

Replace YOUR_API_KEY with the consumer api_key your Impacts Foundry account manager issued. Your domain must be registered as a widget origin against that key (so the CORS check passes) — request additional origins via support.

2. Configuration

Every option is a data-* attribute on the host <div>:

AttributeDefaultPurpose
data-api-keyrequiredYour consumer api_key
data-basehttps://www.impacts-foundry.comOverride only for testing
data-stateall (planning + execution + recent done)Comma-separated subset, e.g. planning,execution
data-scaleallComma-separated filter: micro,mid,macro,borderless
data-limit6Maximum cards to render. Max 100 per API response.

3. Multiple widgets on one page

Drop more than one [data-gmi-widget] div — each auto-discovers its own attributes. Useful for side-by-side category boards:

<div data-gmi-widget data-api-key="KEY" data-scale="micro" data-limit="3"></div>
<div data-gmi-widget data-api-key="KEY" data-scale="macro,borderless" data-limit="3"></div>

4. Programmatic mount (SPAs / route changes)

If your site is a single-page app and you inject the widget div after initial render, kick a rescan:

window.__GMI_WIDGET__.scan();          // re-discovers any new [data-gmi-widget] divs
window.__GMI_WIDGET__.mount(element);  // or mount a specific element directly

5. What a supporter clicks through to

Every CTA on a project card routes through /api/v1/go.php?ask=NN&api_key=KEY. That endpoint:

  1. Logs the click (anonymised — no raw IPs, no cookies; per the privacy policy)
  2. Attributes the click to your consumer key (so you appear in monthly impact reports)
  3. 302 redirects to the project's external destination (GoFundMe, ACNC, sponsor's own donation page, etc.)

If a project has no money-lane ask configured yet, the widget falls back to a "Learn more" link to the project page on impacts-foundry.com.

Privacy & security notes

  • The widget calls only public endpoints (/api/v1/projects, /api/v1/go.php) with your api_key. No cookies, no user data leaves the host page.
  • Exact coordinates for projects involving minors or vulnerable people never reach the widget — the server enforces precision reduction before responding.
  • All project values are HTML-escaped before render to defend against XSS in sponsor-authored fields.
  • The widget bundle is a single static file with no third-party dependencies.
